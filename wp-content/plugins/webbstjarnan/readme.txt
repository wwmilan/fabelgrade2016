== Description ==

A help plugin for Webbstjarnan wordpress users. This plugin adds a custom dashbord helper and an custom widget to display Webbstjarnans logo.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Unzip folder webbstjarnan.zip
2. Upload folder `webbstjarnan` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
